# ui-lab3

Простой графический тайм-менеджер на PyQt6 для архитектурных лабораторий.

## Что внутри
- `main.py` — запуск приложения с интерфейсом и логикой расписания.
- `document.txt` — описание задания и личной документации.
- `venv/` — виртуальная среда (не добавляется в репозиторий, но может быть пересоздана).

## Требования
- Python 3.10+
- PyQt6 (пакет указан в `venv`, можно установить вручную: `pip install pyqt6`).

## Быстрый старт
```bash
python3 -m venv venv
source venv/bin/activate
pip install -r requirements.txt  # если файл отсутствует, установи PyQt6 напрямую
python main.py
```

## Комментарии
- Интерфейс предлагает функции добавления/удаления занятий.
- Конфигурация расписания хранится в памяти, без внешних баз данных.
