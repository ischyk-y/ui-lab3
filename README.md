# ui-lab3

Легка версія текстового редактора на PyQt6, який демонструє розділення UI (меню, тулбари, форматування) від файлової логіки (збереження/завантаження).

## Структура
- `main.py` - точка входу: створює `QApplication` та відображає `TextEditor`.
- `lab3qt/ui.py` - клас `TextEditor` зі всіма UI-компонентами та зв’язком із файловою логікою.
- `lab3qt/io.py` - обробка директорії `data/` та функції `read_text`/`write_text`.
- `data/document.txt` - зразковий текстовий документ.
- `requirements.txt` - залежності для запуску.

## Запуск
```bash
python -m venv venv
source venv/bin/activate
pip install -r requirements.txt
python main.py
```

## Поради
- `data/document.txt` використовується як за замовчуванням, тому модифікуй його або зберігай свої власні файли через меню «File». 
- Ця структура дозволяє локально тестувати UI без змін бізнес-логіки, а також швидко додати нові сервіси з обробки тексту.
